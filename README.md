##PROJE AÇIKLAMASI
Bu projede python programlama kullanırak, kullanıcıdan alınan bilgiler dahilinde otomatik olarak bir html web sayfası oluşturulmuştur.
Program çalıştığında kullanıcıdan ad-soyad, kısa biyografi bilgisi ve aldığı dersleri ister.
Alınan dersleri liste halinde istemektedir.
Ve HTML etiketleri kullanılarak bir web sayfası oluşturuluyor.
Oluşturulan HTML sayfası, basit bir CSS ile renklendirilmiştir. Ve başlık,paragraf ve liste yapısından oluşmaktadır.
